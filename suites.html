📘 Suites numériques

Bienvenue dans le cours de Mathématiques.

Définition

Une suite numérique est une liste de nombres rangés dans un ordre précis. Chaque nombre est appelé un terme de la suite.

Exemple

2 ; 4 ; 6 ; 8 ; 10...

Cette suite augmente de 2 à chaque terme.

Exercice

Compléter la suite suivante :

5 ; 8 ; 11 ; 14 ; .... ; ....

Correction :
5 ; 8 ; 11 ; 14 ; 17 ; 20



Exercice 2

Soit la suite arithmétique de premier terme U₀ = 3 et de raison r = 5.
Calculer U₁, U₂ et U₃.

Correction :
U₁ = 8
U₂ = 13
U₃ = 18

Formules importantes

Suite arithmétique :
U(n+1) = U(n) + r
U(n) = U(0) + n × r

Suite géométrique :
U(n+1) = q × U(n)
U(n) = U(0) × qⁿ
